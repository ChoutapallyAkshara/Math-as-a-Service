Math-as-a-Service

A Spring Boot REST API platform for **computational mathematics**, covering Linear Algebra, Statistics, and Graph Theory.

## Overview
Math-as-a-Service is a backend platform built with **Spring Boot** that exposes REST APIs for advanced mathematical computations.  

## Features

### Matrix Service (`/matrix`)
- Determinant
- LU & QR Decomposition
- Multiplication
- Inverse
- Transpose

### Statistics Service (`/statistics`)
- Linear Regression (slope, intercept, RÂ²)
- Hypothesis Testing (two-sample t-test)

### Graph Service (`/graph`)
- Connected Components
- Graph Coloring (Greedy)
